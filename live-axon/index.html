live axon
by office ca

a fun tool for visualizing 3D models as oblique axonometric projections.
effect is purely visual, no geometry is harmed in the process :)
built with three.js and glsl.

directions:
Rhino 3dm or generic glb/gltf files only. pan + zoom with mouse or touch.
for best results center model at world origin (0, 0, 0).
input custom hex color to change background.
'grab' downloads hi-res PNG screenshot (with transparency).
